Example NATS packages and deployments.

# Package
Create virtual machine images or other archive types to more easily deploy 
NATS servers in a consistent manner.

## packer
Uses packer to create VMWare, VirtualBox and AWS AMIs.

# Server
Setup necessary resources to deploy a NATS server in different cloud 
environments.

## aws
Reference implementation of a NATS server running in AWS.
